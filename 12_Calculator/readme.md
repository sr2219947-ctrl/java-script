🧮 Simple Calculator --->

Live Demo: https://sr2219947-ctrl.github.io/java-script/12_Calculator/ 

A web-based calculator that performs basic Addition of two number using HTMl, CSS, and Javascript.

🚀Features 

Takes two number inputs from the user
Performs addition operation
Displays result instantly on button click
Clean and responsive UI

🛠️ Technologies Used
HTML
CSS
JavaScript

⚙️ How It Works
User enters two numbers in input fields
Clicks the Submit button
JavaScript reads input values
Converts them into numbers
Adds both numbers
Displays result on the screen

💻 Code Logic (JavaScript)
document.querySelector() → selects button
getElementById() → accesses input fields
Number() → converts input values
textContent → displays result
