# 🎉 Click Confetti

A fun and interactive JavaScript mini project where colorful dots appear on the screen wherever you click. Each dot displays a random message and disappears after a few seconds — creating a confetti-like effect.

---

## 🚀 Features

- 🎯 Dots appear exactly where you click
- 🎨 Random background colors
- 💬 Random messages inside each dot
- ⏱️ Auto-remove after 5 seconds
- ⚡ Lightweight and fast (pure JavaScript)

---

## 🛠️ Tech Stack

- HTML
- CSS
- JavaScript 

---

## 📸 How It Works

1. User clicks anywhere on the screen  
2. A circle (dot) is created at that position  
3. A random color is applied  
4. A random message is displayed  
5. The dot disappears after 5 seconds
